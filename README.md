## Escape Test

> It's easier to ask forgiveness \
> than it is to get permission \
> \- Grace Hopper

| **Markdown** | **Render** |
| :----: | :--- |
| \*\*bold\*\*, \_italic\_ | **bold**, _italic_ |
| \`code\`, \~strike\~ | `code`, ~strike~ |
|em\*\*\*bold\*\*\*en |em***bold***en |

```javascript
import "fmt"
function main() {
   ft.Println("Hello, World!")
}
```

~~~md
```go
import "fmt"

function main() {
   ft.Println("Hello, World!")
```
~~~

```md
~~~md
**bold**, _italic_
`code`, ~strike~
em***bold***en
~~~
```
